🎓 About Ask Anton

Ask Anton is a sample AI agent designed to help you understand artificial intelligence concepts and topics.

📋 Important details and AI transparency notes (opens in new window)

Features

  • Intelligent Search: Searches through a curated knowledge base to find relevant AI concepts
  • AI-Powered Responses: Uses Microsoft Foundry models to generate clear, contextual explanations
  • Voice Input: Ask questions using your voice for a hands-free experience

How It Works

Ask Anton combines semantic search with Microsoft Foundry AI models to provide accurate, educational responses about AI concepts. Your questions are sent to Microsoft Foundry along with relevant context from the knowledge base.

Getting Started

Click the ⚙️ Configure button to set up your Microsoft Foundry connection (endpoint, API key, and deployment). Then try asking questions like "What is machine learning?" or "Explain artificial intelligence".

🔑 Entra ID App Configuration

Setting up Entra ID Authentication

To use Entra ID authentication, you need to register an application in the Azure Portal and configure the appropriate permissions.

Step 1: Register Your Application

  1. Go to Azure Portal
  2. Navigate to Microsoft Entra ID → App registrations
  3. Click + New registration
  4. Enter a name (e.g., "Ask Anton")
  5. Select Accounts in this organizational directory only
  6. Click Register
  7. Copy the Application (client) ID and Directory (tenant) ID from the Overview page

Step 2: Configure Authentication

  1. In your app registration, click Authentication in the left menu
  2. Click + Add a platform → Single-page application
  3. Add Redirect URI: https://microsoftlearning.github.io/ai-apps/ask-azure/
  4. Under Implicit grant and hybrid flows, check:
    • ☑ Access tokens
    • ☑ ID tokens
  5. Click Configure

Step 3: Add API Permissions

  1. Click API permissions in the left menu
  2. Click + Add a permission
  3. Select APIs my organization uses
  4. Search for "Microsoft Cognitive Services" or use Application ID: 7d312290-28c8-473c-a0ed-8e53749b6d6d
  5. Select Azure Cognitive Services
  6. Choose Delegated permissions
  7. Check user_impersonation
  8. Click Add permissions

Step 4: Grant Admin Consent

  1. In the API permissions page, click Grant admin consent for [your tenant]
  2. Click Yes to confirm
  3. Verify the status shows a green checkmark

Step 5: Assign RBAC Role (if needed)

Ensure your user account has the Cognitive Services User role on the Foundry/AI Services resource in Azure.

Note: You may need Global Administrator privileges to grant admin consent. If you don't have these permissions, contact your IT administrator.
